London's 1854 cholera epidemic

D3 Data Visualization by Alero Jakpa (Aina)

Introduction

The goal of this project is to create a visualization of Dr. John Snow's map of the London's 1854 cholera epidemic using D3. The map should contain information about the death population distribution, and identify the pumps around the street. By doing this, I aim to recreate the relationship that John Snow found between the pumps and the population. Next, add additional charts that can interact with the map and provide insight on the cholera epidemic. My first step is to learn D3 and its libraries and get familiar with CSS libraries. Practice graphs such as bar chart, pie chart, scatter plots, and play around with understanding data set types.

Data

To create the map and additional charts, the project provided the following data files.
1. Streets.json: An array of X and Y coordinates that represent the streets of London in 1984.
2. Pumps.csv- Contains a total of 13 pumps around the epidemic area
3. Death_age_sex.csv: Contains the x and y coordinates of the death location, the gender, age and sex of each victim.
4. Death days: Contains death count for each day of the outbreak. Each death in the death_age_sex.csv file is sequentially linked to the death_days file.

Outcome and Design choices


1. Using the streets json file, I created the map of London using D3 line library. To account for color blindness, I used orange blue and black for the map legend.
2. With the death days file, I created the bar chart. The bar chart and the map are interactive. The bar chart is grey by default, and turns red on mouse over.
3. The next set of graphs are pie charts- one for age group and the other for gender. Both charts are interactive. When you select any age group, you will see the gender distribution on the right.
